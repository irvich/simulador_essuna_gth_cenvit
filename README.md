# CENVIT - Simulador de Examen

Simulador para Oficiales de Administración.

## Reglas implementadas

- 20 preguntas aleatorias por intento.
- Calificación sobre 20 puntos.
- Aprobación mínima: 14/20.
- Colores institucionales: azul marino, celeste y dorado.
- Logo CENVIT en el encabezado.
- Banco total: 130 preguntas.

## Cómo ejecutar

```bash
npm install
npm run dev
```

Luego abre la URL que indique Vite.

## Archivos principales

- `src/App.tsx`: interfaz y lógica del simulador.
- `src/questions.ts`: banco completo de preguntas.
- `public/logo-cenvit.png`: logo usado en la app.
